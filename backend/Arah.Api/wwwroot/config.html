Configuração

Guia rápido para alternar entre InMemory e Postgres.

Sem persistência automática nesta página

Seleção do provider

Defina o provider desejado em appsettings.json ou variáveis de ambiente.

{
  "Persistence": {
    "Provider": "InMemory" // ou "Postgres"
  }
}

Conexão com Postgres

Configure a connection string quando Provider for Postgres.

{
  "ConnectionStrings": {
    "Postgres": "Host=localhost;Database=araponga;Username=araponga;Password=senha"
  }
}

Variáveis de ambiente (exemplo)

PERSISTENCE__PROVIDER=Postgres
CONNECTIONSTRINGS__POSTGRES=Host=localhost;Database=araponga;Username=araponga;Password=senha

Fluxo recomendado

  1. Use InMemory para testes rápidos e local.
  2. Altere para Postgres em ambientes com banco disponível.
  3. Reinicie o serviço após ajustar a configuração.
